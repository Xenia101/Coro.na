# Coro.na

A Web **Map** Service for the **Corona-virus**

Data : [질병관리본부](http://www.cdc.go.kr/index.es?sid=a2)

Map API : [Kakao Map](http://apis.map.kakao.com/)

## Execution / Test Environment

- Window 10
- Node.js **latest version**
